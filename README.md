# Chimney

A lightweight and performant modpack with quality-of-life features.

![Supports Fabric](https://raw.githubusercontent.com/intergrav/devins-badges/v3/assets/cozy/supported/fabric_vector.svg)

## About

Chimney focuses on performance improvements and essential quality-of-life additions for Minecraft. The pack maintains vanilla gameplay while providing better framerates and reduced lag.

## Mod List I am aiming to include in all versions

| Mod | Category | Description |
|-----|----------|-------------|
| [Sodium](https://modrinth.com/mod/sodium) | Performance | Rendering optimization |
| [Lithium](https://modrinth.com/mod/lithium) | Performance | Server and game logic improvements |
| [FerriteCore](https://modrinth.com/mod/ferrite-core) | Performance | Memory usage reduction |
| [ModernFix](https://modrinth.com/mod/modernfix) | Performance | Startup time and runtime improvements |

| [Entity Culling](https://modrinth.com/mod/entityculling) | Performance | Culls entities that aren't visible |
